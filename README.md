# FireArm Background Checks


This purpose of this repository is for code-sharing amoung members of a DS406 Topics in Data Analytics group project. 


The data required for our analysis is available on the FBI’s NICS but we downloaded it from the Buzzfeed News GitHub repository which converted the data from PDF to 
CSV format (https://github.com/BuzzFeedNews/nics-firearm-background-checks). The data covers firearm background checks from November 1998 – September 2023 by month, 
state, and type.


The aim of this project is to answer the question “Does the rate of change in total firearm background checks over time vary across states?”, alongside generating 
insightful visualisations.
